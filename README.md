# AI Travel Companion & Co-Traveller Matching Platform  
*(VisaVerse Hackathon)*

## Overview
VisaVerse is an AI-powered travel companion designed to help solo travellers
find compatible co-travellers and understand cultural etiquette and
documentation requirements for different countries.

This project is being developed as part of the **VisaVerse hackathon**.

## Problem Statement
Solo travellers often struggle with:
- Finding trustworthy and compatible co-travellers
- Understanding cultural norms in unfamiliar countries
- Navigating travel documentation requirements

## Solution
The platform provides:
- Preference-based co-traveller matching
- AI-generated cultural etiquette guidance
- Country-specific travel documentation insights

## Tech Stack
- Python
- Streamlit
- AI / ML models (matching & content generation)
